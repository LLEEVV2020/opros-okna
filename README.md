# opros-okna
Опросник окон

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
## Усановлено в сборник 

``` bash
# install start
npm install --save axios vue-axios vue-resource vue-router vuelidate vuex
```
